# Witch Hunt: Apocalipsis Mágico 🎮🧙‍♀️

## 📖 Descripción
Estás atrapado en un mundo apocalíptico lleno de magia oscura. Las brujas te persiguen sin descanso y tu misión es **sobrevivir el mayor tiempo posible** mientras eliminas a todas las que puedas para lograr el puntaje más alto.  
Cada vez que acumulas más eliminaciones, las brujas se vuelven más rápidas y peligrosas.

---

## 🚀 Cómo jugar
- **Movimiento:** Usa las teclas `W`, `A`, `S`, `D` para moverte.
- **Disparo:** Haz clic con el mouse para disparar.
- **Recargar:** Presiona `R` para recargar tu arma.
- **Objetivo:** Sobrevive, elimina brujas y acumula puntos.
- **Dificultad:** Cada 5 eliminaciones, la velocidad de las brujas aumenta.

---

## ⚙️ Características principales
- Mundo 3D creado con **Three.js**.
- Sistema de colisiones y físicas con **Octree** y **Capsule**.
- Animaciones de enemigos con **FBXLoader**.
- Sonidos de fondo, disparo, recarga y fin de juego.
- Sistema de puntaje y récord guardado en `localStorage`.
- Menú de pausa y fin de juego integrados en la interfaz.
